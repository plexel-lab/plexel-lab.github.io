お問い合わせ

不具合のご報告・機能のご要望は、お気軽にご連絡ください。

📮 お問い合わせフォームを開く

通常2〜3日以内に確認しています（返信が必要な場合はフォームに連絡先をご記入ください）。

クイック記録と詳細入力の違いは？

「クイック記録」は得点ボタンをタップするだけの試合中向けモードで、次のサーバーも自動表示されます。「詳細入力」はコート上の選手をタップしてサーブ・レセプション・スパイク・ブロックをラリー単位で残すモードです。併用も、どちらか一方だけでも使えます。

試合中はクイックだけ記録して、あとから詳細を入力できますか？

できます。ホームから試合を開いて詳細入力に切り替えると、クイック記録の得点の流れと突き合わせながら入力できます。1点ずれた時点で「第◯セット◯点目で相違」と表示されるので、入力ミスにすぐ気づけます。

記録したデータはどこに保存されますか？

入力中に自動でクラウド（Firebase）に保存されます。体育館などで電波が不安定でも端末内に記録され、通信が回復すると自動で同期されます。画面上部の「保存済み / 同期中 / 未同期」バッジで状態を確認できます。

チームでデータを共有するには？

「チーム」タブからチームを作成すると6桁の招待コードが発行されます。メンバーがコードを入力して参加すると、記録した試合をチーム全員が閲覧できます。誰かが記録中の試合は他のメンバーは閲覧のみになり、記録がぶつかる心配はありません。

確定スコアを手で直せますか？

直せます。クイック記録画面の「確定スコア」の数字をタップして編集してください。手修正した値が最優先で使われ、「✎ 戻す」で自動計算に戻せます。

マイスタッツとは？

自分の背番号を登録すると、直近試合の自分の成績・通算成績・成長グラフをホームからすぐ見られる機能です。無料でお使いいただけます。

RallyLog Pro では何ができますか？

試合レポートPDF（スコア・チーム/選手別スタッツを1枚に）と、CSVデータ出力（Excelで自由に分析）が使えるようになります。記録・チーム共有・ボックススコア・成長分析は無料のままです。

サブスクリプションの解約方法は？

iPhoneの「設定」→ 一番上のApple ID → 「サブスクリプション」→ RallyLog Pro を選んで解約できます。解約後も期間終了までPro機能を利用できます。

相手チームの分析はできますか？

アプリの設定で「相手チーム分析」をオンにすると、相手の決定を記録するときに相手選手の背番号を任意入力できます。スコア画面に「相手の得点源」が表示され、次戦の対策に使えます。

文字が小さくて見づらいです

ホーム右上の歯車アイコン → 設定の「文字サイズ」で「大きめ」「特大」に変更できます。

アカウントを削除したい

ホーム右上の歯車アイコン → 設定 → 「アカウントを削除」から削除できます。個人の試合データは完全に削除され、チームに共有済みの試合はチームオーナーに引き継がれます。この操作は取り消せません。

間違えて記録してしまいました

直前の記録は「1つ戻す」（詳細入力では「取り消し」）でいつでも取り消せます。詳細入力ではセット単位で入力をやり直すこともできます（パネル下部の「詳細入力のやり直し」）。

動作環境

iOS 17.0 以降の iPhone / iPad に対応しています。クラウド保存・チーム共有にはインターネット接続が必要です（記録自体はオフラインでも可能）。
